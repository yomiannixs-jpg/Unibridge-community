# Stage 4A: Real Moderator Powers

Adds functional moderation screens:

- /moderation/reports
- /moderation/moderators
- /moderation/safety

Features:

- Clickable Moderation Center cards
- Report queue with Open, Reviewing, Resolved, Dismissed statuses
- Manual report form
- Clear resolved/dismissed reports
- Moderator assignment UI
- Activate/deactivate moderators
- Safety settings for blocked words, anti-agent mode, required country/deadline fields, repeated-post flagging, and hourly posting limits
- Local-storage persistence for safe frontend-first behavior

No backend/API changes are required.
